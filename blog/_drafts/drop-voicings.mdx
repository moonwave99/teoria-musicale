If you drop any note from a closed voicing down an octave, you get a **drop voicing**.

<Truncate/>

## Drop 2 voicings

Check a regular <Chord>Dm7</Chord> for instance:

<Piano notes="D3 F3 A3 C4" />

If we drop the **second highest note** down one octave, we get the corresponding **drop 2** voicing:

<Piano notes="A2, D3 F3 C4" />
